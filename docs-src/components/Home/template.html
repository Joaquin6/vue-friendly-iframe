vue-friendly-iframe

Vue.js component for creating dynamic async iframes.

View on GitHub Download .zip Download .tar.gz

Installation

npm install vue-friendly-iframe --save-dev

1) Download from NPM

Usage

1) Register the Vue Friendly Iframe component: 

import Vue from 'vue';
import VueFriendlyIframe from 'vue-friendly-iframe';

Vue.component('vue-friendly-iframe', VueFriendlyIframe);

2) In your HTML add the component: 

<vue-friendly-iframe :src="example1Form.src"></vue-friendly-iframe>
